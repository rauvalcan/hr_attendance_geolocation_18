HR Attendance Geolocation

Record employee check-ins and check-outs with precise GPS location.

Key Feature: This module enhances Odoo's standard attendance functionality by automatically capturing and storing the employee's GPS coordinates at the moment of check-in and check-out. It also allows defining specific work locations to automatically verify if the employee is on-site.

Main Features

  • Automatic Geolocation: Captures latitude and longitude from the browser when an employee checks in or out.
  • Work Locations: Define official work sites (e.g., "Main Office", "Warehouse") with GPS coordinates and a valid radius.
  • On-Site Verification: The system automatically cross-references the employee's location with defined work sites to confirm if they are within the designated area.
  • Seamless Integration: Integrates directly into the standard "My Attendances" widget. No extra steps for the employee.
  • User-Friendly Notifications: Provides clear success or error messages to the user (e.g., location captured, permission denied).

How it Works

When an employee clicks the check-in/out button, the browser will prompt for location access (only the first time). Once approved, the module sends the coordinates to Odoo, which saves them in the attendance record and checks if the location matches any pre-configured work sites.

1. Employee checks in via the standard widget.

2. GPS data and Work Location are saved in the attendance record.

Configuration

To define work locations, go to:
Attendances App → Configuration → Work Locations.
